# Career Signal Lab Showcase Submission

## Project headline

Career Signal Lab: an explainable AI tool that turns job goals into proof-of-work projects.

## Short description

Career Signal Lab helps job seekers decide what AI project to build for a target role. It reads profile notes and job-post signals, scores the strongest skill evidence, recommends a project angle, and generates a recruiter-ready pitch.

## What I built

I built a responsive web app with a local analysis engine, sample career paths, a visual signal map, skill-evidence scoring, build plans, AI-method recommendations, and markdown export.

## Why it matters

Many students are told to "build a project" but are not told how to choose one that employers can understand. Career Signal Lab turns career goals into a concrete project plan and makes the reasoning visible.

## AI skills shown

- Natural-language signal extraction
- Recommendation logic
- Explainable scoring
- Responsible AI evaluation
- Product thinking
- Data visualization
- Clear recruiter-facing storytelling

## Responsible AI notes

The prototype runs locally and does not send user profile text to a third-party service. It uses transparent keyword and scoring logic so users can inspect the recommendation rather than treating it as a black box.

## Future version

The next version would use embeddings to compare profile evidence with a larger set of job descriptions, then use an LLM to generate more personalized case-study drafts while keeping the scoring rubric visible.
