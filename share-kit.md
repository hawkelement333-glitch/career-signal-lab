# Career Signal Lab Share Kit

## Handshake or LinkedIn post

I built Career Signal Lab, an explainable AI portfolio tool that helps job seekers turn a target role into a proof-of-work project.

The app compares profile notes with job-post signals, scores skill evidence, recommends a project angle, and generates a recruiter-ready pitch. I wanted the AI behavior to be transparent, so the prototype shows its scoring logic, signal map, build plan, and responsible AI notes instead of hiding the recommendation in a black box.

Skills shown: natural-language signal extraction, recommendation logic, dashboard design, responsible AI, and product storytelling.

## Resume bullet

Built Career Signal Lab, a responsive AI portfolio web app that analyzes role requirements and profile notes to generate explainable project recommendations, visual skill-evidence scoring, and recruiter-ready showcase pitches.

## 30-second demo script

"Career Signal Lab helps students decide what AI project to build for a target role. I paste in a role, profile notes, and job-post signals. The app extracts skill evidence, scores the strongest signals, recommends a project angle, and creates a showcase pitch. I designed it to be transparent, so users can see the score, the skill map, the build plan, and the responsible AI considerations before they use the recommendation."
